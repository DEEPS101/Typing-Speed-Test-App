# Typing Speed Test App

*App is under construction*


## Run the following commamnds in your terminal to get started with Typing Speed Test App 
### To install pygame run:
`pip install pygame`

### To install pyjokes run:
`pip install pyjokes`

### To run the app:
`python typingTest.py`
